# TugasBesar_URO_Programming
~~~
Overview Program: Mekanisme Kapal
1. Program ini bertujuan untuk mengimplementasikan mekanisme kapal dalam berlayar, menyerang, dan bertahan dalam sebuah permainan sederhana. Pada program ini, kapal dapat membaca dan menampilkan posisi saat ini, serta dapat berlayar di empat kuadran pada koordinat kartesian. Kapal dapat bergerak satu satuan ke depan, belakang, kanan, atau kiri dan tidak dapat keluar dari peta yang telah ditentukan.

2. Selain itu, kapal memiliki atribut health yang akan menentukan seberapa banyak serangan yang dapat diterima sebelum kapal kalah. Kapal juga dapat menyerang musuh dengan damage yang konstan dan memiliki jarak tembak maksimum untuk menyerang lawan.

3. Pada awal program, kapal berada pada koordinat (0,0) dan harus menghadapi satu armada musuh yang muncul secara acak di koordinat yang berbeda dengan posisi kapal saat ini. Kapal akan diminta untuk memilih perintah apakah ingin menembak, bergerak, atau diam di tempat. Jika kapal musuh berada di luar jarak tembak kapal maksimum, program akan mengeluarkan pesan error.

4. Program akan berakhir ketika health kapal kurang dari atau sama dengan 0. Pada akhir program, jumlah kapal musuh yang berhasil dikalahkan akan ditampilkan.
~~~

~~~
Cara menjalankan program
1.	Pada awal program akan ditampilkan koordinat kapal dan koordinat musuh
2.	Lalu memilih 3 opsi yaitu bergerak, menembak dan diam di tempat
a.	Jika memilih bergerak :
i.	Akan diminta untuk memilih bergerak 1 satuan kemana, apakah atas, bawah, kiri atau kanan
b.	Jika memilih menembak :
i.	Kapal akan saling menembak dan saling mengurangi HP
c.	Jika memilih diam :
i.	Kapal akan terkena serangan dan hp akan berkurang tanpa mengurangi hp musuh
3.	Program akan berulang sampai hp kapal sendiri habis
4.	Jika program selesai, akan ditampilkan jumlah kapal yang ditenggelamkan
